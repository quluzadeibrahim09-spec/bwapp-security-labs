# bWAPP Security Labs

This repository contains hands-on web application security testing labs performed on a deliberately vulnerable application (bWAPP).


## Environment
- Host OS: macOS (Apple Silicon M2)
- Virtualization: UTM
- Guest OS: Ubuntu
- Application: bWAPP (Buggy Web Application)

## Scope
The purpose of this project is to practice and document common web application security vulnerabilities aligned with the OWASP Top 10.


## Covered Topics
-SQL Injection
-Cross-Site Scripting (XSS)
-Cross-Site Request Forgery (CSRF)
-Command Injection
-Authentication and Authorization Flaws

## Disclaimer
This project is for educational purpose only. No real-world systems were targeted.
